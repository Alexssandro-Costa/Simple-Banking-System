# Projeto de sistema bancario simples
___

Esse projeto tem como objetivo a criação de um sistema bancario simples utilizando ***persistência em arquivos*** utilizando a API JAVA ***NIO.2***. 

> ## Objetivo 
> - Compreender os metodos e classes da API ***NIO.2***;
> - Entender leitura e escrita em arquivos; 
> - Entender persistência de dados;
> - Comprender a criação de objetos por meio de dados recuperados de arquivos;
> - Comprender a escrita organizada de dados;
